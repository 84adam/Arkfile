Arkfile Vault Features

Client-Side Encryption

All files are encrypted in your browser before upload. Your data is protected by AES-256-GCM encryption, ensuring that even admins cannot access your files.

OPAQUE Authentication

Advanced password authentication using the OPAQUE protocol. Your password never leaves your device, providing perfect security even against server compromises.

Mandatory 2FA

Time-based One-Time Passwords (TOTP) are required for all accounts, with support for Google Authenticator, Authy, and other standard 2FA apps.

Secure File Sharing

Share files with additional password protection using Argon2id key derivation. Recipients don't need accounts - just the link and the share password.

Flexible Storage

Built on MinIO with support for S3-compatible storage providers including Amazon S3, Backblaze B2, Wasabi and Vultr Object Storage.

Redundant Backups

Your encrypted files are safely stored with built-in redundancy and replication across multiple drives and storage servers.
